# College Research Assistant

Small research assistant utilities for college projects.

Files
- `a.py` — (placeholder)
- `chat_prompt.py` — chat prompt utilities
- `chatbot.py` — chatbot implementation
- `main.py` — entry point

Usage
1. Create a Python environment (optional):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

2. Run the app:

```powershell
python main.py
```

Notes
- No external dependencies listed. Add a `requirements.txt` if needed.
